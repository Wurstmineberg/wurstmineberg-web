The Wurstmineberg Minecraft Server

Wurstmineberg is a whitelisted vanilla Minecraft server, and a subproject of the Wurstmannberg meta label, which doesn't really mean anything anymore. Come to think of it, it never really did. Oh well, now it does.

The server usually runs the latest version of Minecraft, which means we also try to run the latest stable snapshots. Therefore we can't really use awesome things like bukkit. We also don't run any serverside mods for that reason …Sorry. But we do have some awesome selfmade tools that really replace lots of that functionality.

If you're interested in playing with us, we have some requirements. In the meantime, you can check out the IRC and Twitter section below.

IRC

We use the bleeding edge of internet technology. That's why our primary means of communication is an IRC channel on Freenode. Feel free to join us there if you're interested in playing on the server or just generally want to say hi and / or talk about… Stuff?

Twitter

We integrated Twitter in our server/IRC bot and now we have automated tweets. Sometimes it's a death message, sometimes it's the server telling us that he's lonely, sometimes it's a list of currently online players, sometimes a snippet from IRC shenanigans. It's emotional.

It's also the easiest way to get in touch with us.

Custom Tools

We have a github organization where we keep most of our stuff, like the IRC Bot, the reworked init.d script for easy management, and even this website. We also have two single-serving sites for the current server time and weather conditions (Javascript required). There's also the wurstmineberg APU, which makes a lot of cool stuff possible, like our statistics page.

Teamspeak

How cliché, right? Yes, kind of. But everything's better than group Skype call bulshittery and the likes. Because sometimes it's just nice to talk to people. It's experimental right now, but someone has to test it.

You can get the password on the IRC and the client, if necessary, over there.

Wiki

We set up a MediaWiki to harvest the glorious benefits of wiki-technology for our purposes. While the subreddit is an okay place to discuss things, we can't put everything into Google Docs which get abandoned/forgotten/ignored over time.

Besides meta stuff, we're also writing articles about projects on the server itself. This might not seem like a big deal as of now, but things are slowly coming together, and eventually we'll have documented all the things.

Subreddit

We have a restricted subreddit (meaning: "read only" for strangers). We created it because we thought it would be a good place to update everyone on, well, everything. It's not used anymore because most of its functionality has been replaced by the wiki, but it still serves as a record of server history. You'll find attempts to organize events, announcements, and lots of Google docs.
