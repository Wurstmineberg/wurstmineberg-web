The Wurstmineberg Minecraft Server

Wurstmineberg is a whitelisted vanilla Minecraft server, and a subproject of the Wurstmannberg meta label, which doesn't really mean anything anymore. Come to think of it, it never really did. Oh well, now it does.

The server usually runs the latest version of Minecraft, which means we also try to run the latest stable snapshots. Therefore we can't really use awesome things like bukkit. We also don't run any serverside mods for that reason …Sorry.

If you're interested in playing with us, we're currently working on some guidelines. In the meantime you can check out the IRC and Twitter section below.

Overview map

We set up the Minecraft Overviewer to generate a daily overview map of our world. It tends to drain a lot of resources, so we apologize for not running it every 10 minutes or so, and to ensure it contains the least amount of suck, make sure to tell us where you are or make the pull request yourself.

We use the bleeding edge of Internet technology. That's why our primary means of communication is an IRC channel on Freenode. Feel free to join us there if you're interested in playing on the server or just generally want to say hi and / or talk about… Stuff?

Subreddit

Organizing events and announcing changes to our server is easier in a spot like our restricted subreddit (meaning: "read only" for strangers). You'll find attempts to organize events, announcements and probably some other stuff on the long run. We thought it would be a good place to update everyone on, well, everything.

Twitter

We integrated Twitter in our server/IRC bot and now we have automated tweets. Sometimes it's a death message, sometimes it's the server telling us that he's lonely, sometimes it's a list of currently online players, sometimes a snippet from IRC shenanigans. It's emotional.

It's also the easiest way to get in touch with us.

Custom Tools

We have a GitHub organization where we keep most of our stuff. Fenhl's IRC Bot is already up, and the reworked init.d script for easy management is coming up soon, too. Feautures include Minecraft and IRC chatsyncing, Twitter integration and the countless tools you see on this page. (Javascript required) We also have two single-serving sites for the current server time and weather conditions (forecast coming soon!)

Wiki

We set up a MediaWiki to harvest the glorious benefits of wiki-technology for our purposes. While the subreddit is an okay place to discuss things, some things just need a broader and more collaborative space to be planned, documented or collected. We can't put everything into Google Docs which get abandoned/forgotten/ignored over time. The Wiki might not seem like a big deal as of now, but as time passes and we expand, it can only be helpful.
